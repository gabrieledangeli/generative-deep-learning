# Generative Deep Learning

Implementations of the foundations behind modern generative models, developed as executable notebooks.

## Notebooks

- [01 — Discrete latent variables](notebooks/01_discrete_latent_variables.ipynb):
  K-means, vector quantization, Gaussian-mixture EM, ancestral sampling, and the zero-variance GMM/K-means connection, implemented from scratch with NumPy.
- [02 — Continuous latent variables](notebooks/02_continuous_latent_variables.ipynb):
  PCA, whitening, probabilistic PCA, EM, factor analysis, ICA, Kalman filtering, ELBO diagnostics, nonlinear latent models, and decoder outputs, implemented from scratch with NumPy.
